The project

About New Wave

New Wave is a free, community-driven knowledge base for the Odin programming language — a simple, fast, and modern systems programming language created by Bill Hall (GingerBill).

Why this site?

Odin is a language that deserves excellent learning resources. While the official documentation is solid, beginners and intermediate programmers often need approachable guides, worked examples, and practical tutorials to really understand a language. That's what New Wave aims to provide.

What's here

All content is free and will remain free. There are three main content types:

  • Blog — articles on language features, patterns, performance, and ecosystem news.
  • Docs — reference documentation written to complement the official spec.
  • Tutorials — hands-on guides from "Hello, World!" to complex real programs.

How it's built

This site is intentionally lightweight. Everything runs on Bun.js serving static HTML, CSS, and vanilla JavaScript. Content is stored as plain Markdown files. No database, no heavy framework, no tracking. The whole site loads in under 100ms on a modern connection.

Markdown is parsed client-side using marked.js and code blocks are highlighted using highlight.js.

Contributing

All content lives in the content/ folder as Markdown files with YAML frontmatter. To add or improve content, write a .md file in the appropriate folder (blog/, docs/, or tutorials/) and follow the frontmatter format used by existing files.

Frontmatter reference

Each Markdown file can include the following frontmatter fields: 

---
title: Your Post Title
date: 2026-01-15
description: A short one-sentence description.
author: Your Name
tags: odin, beginner, syntax
readTime: 8 min read        # for blog posts
difficulty: beginner        # for tutorials
duration: 20 minutes        # for tutorials
order: 1                    # for docs (controls sidebar order)
---
